## Troubleshooting

### "Revenue not updating"
**Solution:** Refresh the page. Revenue updates every 15 minutes.

### "Content stuck in pending"
**Solution:** Wait 2-4 hours. If still pending after 24hrs, contact support.

### "Can't see recent uploads"
**Solution:** Click "View All" to see complete library. Dashboard shows last 3 only.

### "Dashboard loading slowly"
**Solution:**
- Clear browser cache
- Check internet connection
- Try different browser
- Disable browser extensions temporarily
