## Setting Up Your Profile

Your profile is your storefront - it's the first impression potential subscribers see.

**Step 1: Navigate to Settings**

- Display Name

**Requirements:**
- Minimum size: 400√ó400 pixels

## Privacy Settings

Control who sees your content and profile.

### Go to Settings ‚Üí Privacy

### Profile Visibility

**Public Profile** (Default)
- Anyone can view your profile
- Shows profile photo, banner, bio
- Shows subscriber count
- Does NOT show subscriber-only content

**Private Profile**
- Only approved followers see profile
- You must approve each follower request
- Better for exclusive/niche audiences

**How to Enable:**
Settings ‚Üí Privacy ‚Üí **Private Profile** toggle

### Content Visibility

**Subscriber Content:**
- Always private (only active subscribers can view)
- Cannot be made public

**Free Posts:**
- Set individual posts to public or subscribers-only
- Choose when uploading each post

### Search & Discovery

**Allow Search Engines:**
- Toggle ON: Your public profile appears in Google searches
- Toggle OFF: Only accessible via direct FanzUncut link

**Recommendation:** Keep ON for maximum discoverability

**Appear in Recommendations:**
- Toggle ON: Your profile shown to potential subscribers
- Toggle OFF: Only accessible via direct link

**Recommendation:** Keep ON to grow your audience

### Messaging Privacy

**Who Can Message You:**
- Everyone
- Subscribers Only
- No One

Go to Settings ‚Üí Privacy ‚Üí **Message Settings**

**Message Requests:**
- Non-subscribers send "message request"
- You approve or decline
- Prevents spam

### Story Privacy

**Who Can View Stories:**
- All Subscribers
- VIP/Premium Tier Only
- Select Subscribers (manually choose)

## Security Settings

Protect your account from unauthorized access.

### Two-Factor Authentication (2FA)

**Why Enable:**
- ‚úÖ Prevents unauthorized access
- ‚úÖ Required for high-earning accounts ($1000+/month)
- ‚úÖ Protects your earnings and content

**How to Enable:**

1. Settings ‚Üí Security ‚Üí **Two-Factor Authentication**
2. Choose method:
   - **Authenticator App** (recommended): Google Authenticator, Authy
   - **SMS**: Text message codes
   - **Email**: Email codes

3. For Authenticator App:
   - Download Google Authenticator or Authy
   - Scan QR code shown on screen
   - Enter 6-digit code from app
   - Save backup codes (write down and store securely)

4. Confirm setup

**Login with 2FA:**
1. Enter username/password
2. Enter 6-digit code from app
3. Access granted

**Lost Access:**
Use backup codes saved during setup

### Active Sessions

**View Active Sessions:**

**Shows:**

- IP addresses
- Last activity time
- Browser/app used

**Log Out Remotely:**
- Click **End Session** next to any device
- Logs out that device immediately
- Use if you left yourself logged in somewhere

**Log Out All Devices:**
Settings ‚Üí Security ‚Üí **Log Out All Sessions**
- Logs out every device except current one
- Use if account compromised

### Login History

**View Login History:**
Settings ‚Üí Security ‚Üí **Login History**

- Login date/time

**Suspicious Activity:**
If you see logins you don't recognize:

## Troubleshooting

### Common Issues

**"Username already taken"**
- Usernames are unique across platform
- Try variations: add numbers, underscores
- Check if someone is impersonating you (report if so)

**"Cannot change username"**
- Can only change once per 60 days
- Check Settings ‚Üí Profile for next available change date

**"Profile photo won't upload"**
- Check file size (max 10 MB)
- Check format (JPG, PNG, GIF only)
- Ensure dimensions are at least 400√ó400
- Try different browser if issue persists

**"Email already in use"**
- Each email can only be linked to one account
- If it's your email, you may have forgotten you have an account
- Use password reset to recover old account
- Contact support to merge/transfer accounts

**"Cannot verify identity"**
- Ensure ID is clear and readable
- Photo must be in focus
- All four corners of ID visible
- Selfie with ID must show your face clearly
- Try different lighting
- Contact support if repeatedly rejected

**"Not receiving notification emails"**
- Check spam/junk folder
- Add support@FanzUncut.com to contacts
- Verify email in Settings ‚Üí Account
- Check Email Notification settings (may be turned off)

**"Cannot enable 2FA"**
- Check system time on device (must be accurate for authenticator)
- Try different 2FA method (SMS instead of app)
- Clear browser cache and try again
- Contact support
