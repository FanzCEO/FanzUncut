## Host Merchant Services (HMS) Integration

All payments are routed through Host Merchant Services for:
- **Risk Management** - Chargeback monitoring
- **MID Rotation** - Multiple merchant accounts
- **Smart Routing** - Optimal processor selection
- **Backup Processing** - Failover capabilities

## FanzDash Integration

All transaction events are reported to FanzDash for:
- **Centralized Monitoring** - Unified dashboard
- **Payout Orchestration** - Automated creator payouts
- **Compliance Tracking** - Regulatory reporting
- **Performance Analytics** - Business intelligence

## Compliance Verification

Run the payment compliance checker:

```bash
npm run check:payments
```

This will verify:
- No banned processors in codebase
- Adult-friendly processors configured
- HMS integration active
- FanzDash reporting enabled
